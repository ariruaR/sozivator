    "Stop asking. Start commanding."

Это не просто Telegram-бот. Это инструмент социального инжиниринга и абсолютного контроля над твоими чатами. Создан для тех, кому надоело ждать ответа в TikTok-эпоху.
🦾 ВООРУЖЕНИЕ v5.0 (CORE FEATURES)

Бот построен на архитектуре Hybrid Memory (In-Memory Dictionary + JSON Persistence), что делает его неуязвимым к перезагрузкам и спам-атакам.

    🕹️ Stark Admin Panel: Полноценный пульт управления. Запуск рассылок, выгрузка списков "Лузеров" и статистика — всё через Inline-кнопки.
    👁️ Spy Engine (Time Tracking): Бот фиксирует время отправки и ответа с точностью до миллисекунды. Никаких «я не видел». Бот знает всё.
    💣 Digital Siege (Module of Insults): Рандомная генерация унижающих фраз каждые 10 минут для тех, кто игнорирует приказ.
    📓 Persistence JSON: Вечное досье на каждого юзера. Бот помнит каждый косяк.
    🗳️ Interactive Voting: Демократия под твоим контролем. Голосование за место встречи с защитой от дублей.
    🛡️ Show Alert UI: Модальные окна, блокирующие экран пользователя до подтверждения признания вины.

🚀 ПРОТОКОЛ ЗАПУСКА (INSTALLATION)

Если ты не NPC и понимаешь, как работает Python, следуй инструкциям:

    Клонируй репозиторий:

    Bash

    git clone https://github.com/твой_ник/SauronEyeBot.git
    cd SauronEyeBot

    Снаряжение (Requirements):

    Bash

    pip install -U aiogram

    Настройка (Config):
    Открой tgbot.py и впиши свои данные в переменные:
        TOKEN — твой ключ от @BotFather.
        ADMIN_ID — твой числовой ID (узнай его через @userinfobot).

    Арсенал (Assets):
    Положи картинки (rock.png, dora.png и т.д.) в папку с ботом.
